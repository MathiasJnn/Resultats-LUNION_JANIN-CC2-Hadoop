# Resultats-LUNION_JANIN-CC2-Hadoop

Ce dépôt contient les fichiers de résultats obtenus pour l’examen CC2 Hadoop à partir du dataset **MovieLens 25M** (fichier `tags.csv`).

Chaque fichier correspond à une question :

* `q1_resultat.txt` : nombre de tags par film
* `q2_resultat.txt` : nombre de tags par utilisateur
* `q4_resultat.txt` : nombre d’utilisations de chaque tag
* `q5_resultat.txt` : nombre de tags par film et par utilisateur

Les résultats ont été générés à l’aide de scripts **MapReduce (MRJob)** exécutés sur le fichier `tags.csv`.

Les fichiers étant volumineux, ils sont fournis directement dans ce dépôt et sont référencés dans le compte rendu Markdown demandé pour le rendu.
